# SkripsiMuhamadFazri
Repository ini berisi penelitian analisis sentimen publik terhadap Pertamina di platform X menggunakan metode Aspect-Based Sentiment Analysis (ABSA) dan model IndoBERTweet untuk menganalisis persepsi masyarakat berdasarkan aspek Harga BBM, Kualitas BBM, Pelayanan SPBU, Kebijakan Pertamina, dan Umum.
