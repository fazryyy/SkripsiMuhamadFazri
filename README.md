Aspect-Based Sentiment Analysis terhadap Pertamina pada Platform X menggunakan IndoBERTweet

Repository ini berisi skripsi penelitian mengenai analisis sentimen publik terhadap Pertamina pada platform X (Twitter) menggunakan pendekatan Aspect-Based Sentiment Analysis (ABSA) dengan model IndoBERTweet. Penelitian bertujuan untuk memahami persepsi masyarakat terhadap Pertamina berdasarkan aspek tertentu seperti Harga BBM, Kualitas BBM, Pelayanan SPBU, Kebijakan Pertamina, dan aspek Umum.

Deskripsi Penelitian

Penelitian dilakukan menggunakan pendekatan Knowledge Discovery in Database (KDD) yang meliputi:

Data Collection (Crawling Data X/Twitter)
Data Preprocessing
Aspect Extraction
Sentiment Classification
Analysis & Visualization

Dataset dikumpulkan dari platform X selama periode September–Oktober 2025 menggunakan teknik crawling dan diproses menggunakan Python di Google Colab.

Hasil Penelitian

Hasil penelitian menunjukkan bahwa pendekatan ABSA mampu memberikan gambaran distribusi sentimen publik secara lebih spesifik berdasarkan aspek yang dibahas masyarakat. Analisis ini membantu mengidentifikasi aspek yang dominan memperoleh sentimen positif maupun negatif terhadap Pertamina.
